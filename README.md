# TWA (Talk With Anyone)

TWA (Talk With Anyone) is an open-source project focused on connecting users worldwide through lightweight, real-time, and privacy-focused chat spaces. 

Built using **JavaScript (Node.js + Socket.io)** and **HTML/CSS**, TWA offers transient, zero-clutter messaging environments distributed across multiple structural zones.

---

## 📜 Community Rules

To ensure a safe, clean, and welcoming environment for everyone, all users must follow these rules:

1. **English Only:** You may only communicate in English.
2. **No Spamming:** Do not spam. *(Enforced by our built-in 1-minute message cooldown).*
3. **Be Respectful:** Do not harass, threaten, or judge anybody.
4. **Keep it Clean:** No NSFW (Not Safe For Work), religious, and/or political contexts.
5. **Public Spaces:** This is not a private messaging app. Anyone can see and respond to any of your messages.

---

## 🚀 Key Features

*   **Zone & Channel Architecture:** 3 distinct Zones, each hosting 16 separate chat channels (48 rooms total).
*   **Ephemeral Messaging:** Server-side data automatically wipes chat channels clean after 24 hours of inactivity to prevent data overhead.
*   **Spam Protection:** Hard-coded 1-minute global cooldown preventing users from repeating identical messages.
*   **Secure Authentication:** Quick account creation and login powered by secure password hashing.

## 🛠️ Tech Stack

*   **Frontend:** Vanilla HTML5, CSS3, JavaScript
*   **Backend:** Node.js, Express, Socket.io
*   **Database/Caching:** Redis (handles real-time history TTL and spam tracking)
